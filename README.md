## 🔥 What is LuzidSettings?

LuzidSettings is a **powerful, all-in-one Windows optimization suite** with a sleek dark tactical UI. It combines over **67 individual tweaks** into a single polished application — covering everything from disabling telemetry and killing bloat services to Windows 11 UI fixes, live system monitoring, game detection, benchmarking, process scanning, network analysis, and more.

No subscriptions. No telemetry. No bloat. Just raw performance.

> ⚠️ **Run as Administrator** for full functionality. Registry and service tweaks require elevated privileges.

### 📊 Live System Monitor
Real-time CPU and RAM graphs with a 60-second rolling window. Header badges update every 2 seconds with CPU %, RAM %, Disk %, and running process count.

### ⚡ Quick Profiles
One-click optimization presets that apply a full set of tweaks instantly:
- 🎮 **Gaming** — maximum FPS, minimum latency
- 💼 **Work** — balanced performance, stable system
- 🔇 **Silent** — low power, quiet operation
- ⚖️ **Balanced** — sensible defaults for everyday use

Auto game detection switches profiles automatically when a supported game is launched.

### 🗂️ Registry Tweaker
Direct registry editing with pre-built tweak presets. Visual key browser with apply and revert support.

### 🏎️ Benchmark Engine
Built-in CPU, RAM, and disk performance benchmarks. Measure real gains before and after applying tweaks.

### 🚀 Startup Manager
Full control over every program that auto-starts with Windows. Disable and re-enable entries without uninstalling anything.

### 🔍 Process Scanner
Every running process with CPU and RAM breakdown. Kill background bloat with one click.

### 🌐 Network Analyzer
Live network stats and interface information. Refreshes every 4 seconds.

### 🛡️ Restore Point Manager
Create named system restore points before applying tweaks. Fully reversible if anything goes wrong.

### ▲ FPS Overlay
Transparent always-on-top FPS counter. Toggle from the header bar at any time.

## ⬇️ Installation

### Option 1 — Standalone EXE (Recommended)
1. Go to [**Releases**](../../releases/latest)
2. Download `LuzidSettings.exe`
3. Right-click → **Run as Administrator**
4. Done ✅

No Python needed. No installer. Just run.

### Option 2 — Run from Source

**Requirements:**
- Windows 10 or 11
- Python 3.10+

```bash
# Clone
git clone https://github.com/YOUR_USERNAME/LuzidSettings.git
cd LuzidSettings

# Install dependencies
pip install -r requirements.txt

# Run
python debug.py
```

---

## 🛠️ Build from Source (Create EXE)

```bash
pip install pyinstaller

pyinstaller --onefile --windowed --name LuzidSettings --hidden-import customtkinter --hidden-import PIL debug.py
```

Output: `dist/LuzidSettings.exe`

## ⚠️ Disclaimer

Some tweaks modify Windows registry keys and system services. While all tweaks are designed to be reversible, **always create a restore point before applying changes** — the built-in restore point manager does exactly this.

- Disabling Windows Defender is only recommended in isolated environments
- SMBv2 disable should only be used if network file sharing is not required
- VBS/HVCI changes and HPET changes require a reboot to take full effect

**Use at your own risk. The authors take no responsibility for system instability or data loss.**
