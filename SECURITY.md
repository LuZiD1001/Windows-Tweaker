# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in LuzidSettings, please **do not** open a public issue.

Instead, open a GitHub Security Advisory or contact the maintainer directly.

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You will receive a response within 72 hours.

## Scope

LuzidSettings modifies Windows registry keys and system services. All changes are intended to be user-initiated and reversible. The application does not make any network requests, does not collect data, and does not phone home.
